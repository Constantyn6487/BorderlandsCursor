## What's New in v1.0.0 🚀
Initial release of the fan-made Borderlands cursor packs for Windows!

### Inside the archive:
* **BL_Cursors_128** — Full 128px high-resolution cursor theme (Standard style)
* **BD_Cursors_128** — Full 128px high-resolution cursor theme (Alternative style)
* **4Cursors_64** — Lightweight pack with 4 essential cursors scaled down to 64px

*Disclaimer: A non-commercial Windows cursor pack inspired by the Borderlands universe. Perfect for fans of the game who want to customize their desktop.*

## 📥 Installation
1. Download the cursor archive.
2. Extract the files to a convenient folder (for example, `C:\Windows\Cursors\Borderlands`).
3. Right-click the `install.inf` file and select **Install**.
4. Open the Windows Mouse Properties and select the "BL_BorderlandsPack or BD_BorderlandsPack" scheme.

## Что нового в версии v1.0.0 🚀
Релиз фанатского набора курсоров Borderlands для Windows!

### Что внутри архива:
* **BL_Cursors_128** — Полный накладной пак курсоров высокого разрешения 128px (Стандартный стиль)
* **BD_Cursors_128** — Полный накладной пак курсоров высокого разрешения 128px (Альтернативный стиль)
* **4Cursors_64** — Облегченный пак из 4 основных курсоров, уменьшенных до 64px

*Disclaimer: Это некоммерческий фанатский набор курсоров для Windows, вдохновленный вселенной Borderlands. Подходит для фанатов игры, желающих кастомизировать свой рабочий стол.*

## 📥 Установка (Installation)
1. Скачайте архив с курсорами.
2. Распакуйте файлы в удобную папку (например, `C:\Windows\Cursors\Borderlands`).
3. Кликните правой кнопкой мыши по файлу `install.inf` и выберите **Установить** (Install).
4. Откройте параметры мыши в Windows и выберите схему "BL_BorderlandsPack или BD_BorderlandsPack".
